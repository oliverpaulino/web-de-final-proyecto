Nota de Implementación (Proxy Javalin)

Este cliente consume los endpoints REST proxy configurados en Javalin. Javalin se encarga de crear el canal y llamar a los stubs de gRPC internamente. El flujo exacto es:

Navegador → REST (Javalin :7770/api/surveys-grpc-proxy) → gRPC Stub → Servidor gRPC Java (:9090)
