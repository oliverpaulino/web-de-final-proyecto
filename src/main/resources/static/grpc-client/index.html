Nota de Implementación

Este cliente utiliza gRPC-Web con el proxy Envoy para comunicarse con el servidor gRPC de Java. El flujo es:

Browser → HTTP/1.1 → Envoy Proxy :8080 → gRPC (HTTP/2) → Javalin Server :9090

Los stubs JavaScript se generan con: protoc --js_out=... --grpc-web_out=...
